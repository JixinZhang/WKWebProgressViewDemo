# WKWebProgressViewDemo
iOS WKWebView添加网页加载进度条  

- 微信打开网页时，在导航栏的下方有一个进度条显示网页加载的进度。
有一个进度条显示加载进度会让用户觉得加载的过程更快，并且能够直观的了解到加载到什么程度了，用户体验会更好。

- 详细讲解可前往<a href="http://www.jianshu.com/p/d3c8ba672760">这篇文章</a>
